Project Proposal

Description

Our project is based on HOTEL MANAGEMENT SYSTEM. In this project, we provide many services to the user. The user have to register to the system by providing necessary information. The information will be stored in secure database. Whenever the user want to access system, he/she must login with their registered information. The home page contains necessary information about the system and the services that the system provides.

Features

  1. Room Reservation
  2. Multiple Payment options
  3. Restaurant Reservation
  4. Price Based Hotel Recommendation
  5. Location based Hotel Recommendation

Development Tasks

  1. Gathering Requirements.
  2. Designing the wireframe of the user interface.
  3. Adding Functionality to the interface by adding database and API.

Time Line

  1. Project Proposal (1 November, 2020)
  2. Mid Evaluation (30 November, 2020)
  3. Final Evaluation (30 December, 2020)

Development Cost

No development cost due to availability of free integrated development environment. There can be cost added in the future.
